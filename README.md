## 📌 项目介绍

> 🤖 **特别说明**：本项目的所有代码与文档均完全由 AI 自动生成！
> 💡 **命名彩蛋**：这个极具极客范儿的插件名称 `DOM-inator` 是由 AI 助手亲自挑选的，作者充分尊重并采纳了 AI 的品味与建议！

**💡 项目初衷**：在浏览掘金时，总能看到一些令人不悦的用户（比如讨厌的 "desirepaths 哥"）频繁出现。为了眼不见为净，我们需要一个能灵活屏蔽特定文本，甚至连同整个父级卡片一起隐藏的工具，于是这个专属的“防身”插件就此诞生。

这是一个轻量、强大且可视化的 Chrome 浏览器网页元素屏蔽插件。它可以帮助你自定义屏蔽网页中不需要的内容（如广告、推荐流、特定用户的评论等），为你提供更加清爽纯净的网页浏览体验。

与传统的广告拦截插件不同，本项目专注于**提供直观的可视化选择器**以及**灵活的自定义屏蔽规则**（CSS 选择器、特定文本、图片片段），并且以非破坏性的方式（显示“🚫 此内容已被插件屏蔽”）隐藏元素，方便用户随时了解被屏蔽的情况。

## ✨ 核心功能

*   **侧边栏常驻面板**：点击插件图标会在浏览器右侧打开控制面板，边看网页边操作，绝不打断你的浏览心流。
*   **全局与域名级规则隔离**：支持配置仅在当前域名生效的规则，或对所有网站生效的全局规则，自由切换。
*   **可视化“拾取器”带实时预览**：无需懂代码！点击“🎯 提取/选择”按钮，鼠标悬停在网页元素上时，旁边会实时浮动显示将要提取的特征（文本/CSS/图片链接），点击即可自动生成规则。
*   **四大屏蔽模式**：
    1.  **特定文本屏蔽**：提取网页上的某段文字（如某个杠精的用户名），并可选择屏蔽这段文字所在的“整个评论框”。
    2.  **CSS 选择器屏蔽**：精准定位并屏蔽网页结构（智能优化算法，一键屏蔽所有同类元素）。
    3.  **CSS 选择器 + 包含文本 双重过滤**：精准打击！先用 CSS 圈定一批元素（如所有评论列表），再通过“必须包含的文本”二次过滤，只屏蔽包含特定关键词的那一项。
    4.  **特定图片屏蔽**：点击提取广告图片链接，直接将广告图或整个广告位干掉（完美支持 SVG 图片的智能特征提取）。
*   **单条规则独立开关**：在规则列表中，可以随时拨动开关来开启或暂停某一条具体的屏蔽规则，修改实时生效。
*   **全局一键启停**：顶部设有一个总开关，遇到需要查看原始页面的情况，一键暂停所有屏蔽。
*   **状态智能记忆**：即使关闭侧边栏，你输入到一半的规则和拾取回来的内容也会被自动保存，下次打开依然存在。

## 🚀 安装教程 (开发者模式)

由于本插件暂未上架 Chrome 网上应用店，你需要通过“开发者模式”来手动加载它。整个过程非常简单，只需 1 分钟。

### 第一步：获取插件文件
确保你已经拥有了 `basic-extension` 这个文件夹，并且里面包含了 `manifest.json` 等文件。

### 第二步：打开 Chrome 扩展管理页面
1. 打开你的 Google Chrome 浏览器。
2. 在地址栏输入 `chrome://extensions/` 并按下回车键。
3. （或者点击浏览器右上角的三个点 `⋮` -> 选择 **扩展程序** -> **管理扩展程序**）。

### 第三步：开启“开发者模式”
在扩展管理页面的**右上角**，找到 **开发者模式 (Developer mode)** 的开关，点击将其打开（开关会变蓝）。

### 第四步：加载插件
1. 开启开发者模式后，页面左上角会出现三个新按钮。
2. 点击最左边的 **“加载已解压的扩展程序” (Load unpacked)**。
3. 在弹出的文件选择窗口中，找到并选中你刚才准备好的 `basic-extension` 文件夹，然后点击“选择/确定”。

### 第五步：固定插件（推荐）
1. 加载成功后，你会在页面里看到 **DOM-inator** 的卡片。
2. 点击浏览器右上角的“拼图”图标（扩展程序按钮）。
3. 找到 **DOM-inator**，点击它旁边的 **“图钉”图标**，将其固定在浏览器的工具栏上，方便随时调用。

## 💡 使用指南

1. **打开你要清理的网页**。
2. 点击浏览器右上角刚刚固定的 **DOM-inator** 插件图标，浏览器右侧会弹出控制面板。
3. 在顶部选择你要配置的是 **当前域名** 还是 **全局规则**。
4. 根据你的需要，在下拉菜单中选择屏蔽类型（文本 / CSS / CSS+文本双重过滤 / 图片）。
5. 点击面板上的 **“🎯 提取”** 或 **“🎯 选择”** 按钮。
6. 把鼠标移到左侧的网页中，你会看到一层半透明的黑色遮罩，移动鼠标不仅会高亮对应的元素，鼠标旁边还会**悬浮显示将要提取的预览特征**。
7. 点击你想屏蔽的元素，它的特征信息会立刻回填到右侧的面板输入框中。
8. 在面板中点击 **“添加到当前域名”** 或 **“添加到全局规则”**，底部会有成功提示弹窗，列表自动滚动到底部。
9. 该元素会被瞬间隐藏，并显示“🚫 此内容已被插件屏蔽”。
10. 你可以在下方列表中通过开关独立开启/暂停该规则。

*（如果想更新代码：在代码编辑器里修改代码后，回到 `chrome://extensions/` 页面，点击该插件卡片右下角的 **“刷新” (🔄)** 图标，然后刷新你要测试的网页即可生效。）*
