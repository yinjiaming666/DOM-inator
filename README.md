# DOM-inator (DOM 终结者) 🤖

## 📌 项目介绍

> 🤖 **特别说明**：本项目的所有代码与文档均完全由 AI 自动生成！
> 💡 **命名彩蛋**：这个极具极客范儿的插件名称 `DOM-inator` 是由 AI 助手亲自挑选的，作者充分尊重并采纳了 AI 的品味与建议！

**💡 项目初衷**：在浏览掘金时，总能看到一些令人不悦的用户（比如讨厌的 "desirepaths 哥"）频繁出现。为了眼不见为净，我们需要一个能灵活屏蔽特定文本，甚至连同整个父级卡片一起隐藏的工具，于是这个专属的“防身”插件就此诞生。

这是一个轻量、强大且可视化的 Chrome 浏览器网页元素屏蔽插件。它可以帮助你自定义屏蔽网页中不需要的内容（如广告、推荐流、特定用户的评论等），为你提供更加清爽纯净的网页浏览体验。

与传统的广告拦截插件不同，本项目专注于**提供直观的可视化选择器**以及**灵活的自定义屏蔽规则**（CSS 选择器、特定文本、图片片段），并且以非破坏性的方式（显示“🚫 此内容已被插件屏蔽”）隐藏元素，方便用户随时了解被屏蔽的情况。

## ✨ 核心特性

- 🎯 **可视化拾取元素**：提供类似开发者工具的元素拾取器，点击页面上的任意元素即可自动生成唯一 CSS 选择器并添加到屏蔽规则中。
- 🛠️ **多维度屏蔽规则**：
  - **CSS 选择器屏蔽**：根据 class、id 或层级精准屏蔽特定元素。
  - **文本内容屏蔽**：支持根据元素包含的关键字进行屏蔽。
  - **图片/URL片段屏蔽**：支持屏蔽特定 src 的图片。
- 📦 **支持父容器屏蔽**：对于文本或图片匹配，可以指定向上查找并屏蔽特定的父容器（例如：屏蔽包含“某明星”的整个卡片或评论列表项）。
- 👁️ **非破坏性视觉反馈**：
  - 普通元素被屏蔽后不会导致页面布局严重错位，而是替换为“🚫 此内容已被插件屏蔽”的占位符。
  - 图片/视频被屏蔽后将展示为半透明和模糊状态，避免误杀重要内容。
- ⚡ **动态内容拦截**：内置 MutationObserver 监听，完美支持瀑布流、无限下拉和懒加载元素的实时屏蔽。
- 🌐 **域名级隔离**：所有规则自动按当前域名分组存储，不同网站的规则互不干扰。

## 🚀 安装指南
1. 将本项目下载或 Clone 到本地。
2. 打开 Chrome 浏览器，访问扩展程序管理页面 `chrome://extensions/`。
3. 在右上角开启 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**。
5. 选择本项目的根目录（包含 `manifest.json` 的文件夹）即可完成安装。

## 💡 使用说明
1. 在任意网页中点击浏览器右上角的插件图标，打开管理弹窗。
2. **可视化屏蔽**：
   - 选择“通过 CSS选择器 屏蔽”，点击 🎯 选择 按钮。
   - 在页面中移动鼠标，插件会高亮当前元素。
   - 点击不需要的元素，即可自动生成规则并屏蔽。
3. **内容屏蔽（按文字/图片）**：
   - 选择对应模式，输入需要屏蔽的关键字（如 "推广"）。
   - 若需要屏蔽整个父容器，可在“父容器选择器”中填入对应类名（如 `.feed-card`）。
4. 所有规则即时生效，如果页面有动态加载的新内容也会被自动拦截。
